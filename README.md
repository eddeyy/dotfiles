# Dotfiles

## Setup

### Install all at once
```
$ git clone https://github.com/eddeyy/dotfiles.git
$ cd dotfiles
$ /bin/bash setup.sh
```

### Install Tmux
```
$ /bin/bash /tmux/setup.sh
```

### Install Fish
```
$ /bin/bash /fish/setup.sh
```

### Install Neovim
```
$ /bin/bash /nvim/setup.sh
```
